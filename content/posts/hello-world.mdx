Welcome to the new site! I've migrated from WordPress to a statically generated Next.js blog powered by MDX.

The new stack is simpler, faster, and easier to maintain. Posts are just Markdown files in a Git repo — no database, no plugins, no security patches.

## Why I switched

- **Performance**: Static HTML served from a CDN loads instantly.
- **Simplicity**: No WordPress admin, no PHP, no MySQL.
- **Control**: Every piece of the site is just code I can version and deploy.

## How it works

Each post is an `.mdx` file with frontmatter for metadata. Push to GitHub, Vercel builds and deploys automatically. That's it.
